
# FundRaisingGOGO 募資網站練習
這是一個基於Ruby on Rails所開發的線上募資網站，此網站主要用來熟悉Rails框架的使用，以及如何與Web API、第三方金流串接

## 使用到的Gem

◆ devise                    > 用於快速開發會員系統
◆ devise-i18n               > 建立多國語系
◆ omniauth-google-oauth2    > 用於串接Google Login
◆ carrierwave               > 用於圖檔上傳
◆ rails_admin               > 用於快速建立後台
◆ rails_admin-i18n          > 後台使用多國語系
◆ sunspot_rails
◆ sunspot_solr
◆ dotenv-rails
◆ pg
◆ mysql2
